# Examples

## screenshot.png

Add a screenshot of the HTML data diff report here. To capture one:

1. Run `./data-diff.sh` against a model with real changes
2. Open the generated HTML in your browser
3. Take a screenshot showing the summary + at least one model card
4. Save as `screenshot.png` in this directory
